# Snake Game

A minimal classic Snake implementation using vanilla HTML, CSS, and JavaScript.

## Run locally

1. Start the local server:
   ```bash
   npm start
   ```
2. Open `http://localhost:4173` in your browser.

## Test

```bash
npm test
```

## Manual verification checklist

- [ ] Snake moves continuously on a grid.
- [ ] Arrow keys and WASD both steer the snake.
- [ ] On-screen arrow controls work.
- [ ] Snake grows and score increments after eating food.
- [ ] Game ends on wall collision.
- [ ] Game ends on self collision.
- [ ] Restart button resets score and starts a new run.
